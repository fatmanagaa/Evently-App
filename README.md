# 🗓️ Evently

A Flutter app for discovering, creating, and managing events — with a clean UI, dark/light mode, and multilingual support.

---

## 📖 Description

Evently lets users sign up or log in (including Google Sign-In), browse events by category, add new events with a title, description, date, and time, and save their favourites — all synced in real time via Firebase.

---

## ✨ Features

- 🔐 Email/Password & Google Sign-In authentication
- 🌅 Onboarding screens for first-time users
- 📋 Browse events by category (Sports, Birthday, Meeting, Book Club, Exhibition, and more)
- ➕ Add new events with title, description, date, time, and category image
- ❤️ Save favourite events
- 🌙 Dark & Light theme support
- 🌍 Arabic & English localisation (i18n)
- 🔑 Forgot password flow

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Flutter | UI framework |
| Firebase Auth | User authentication |
| Cloud Firestore | Real-time database |
| Google Sign-In | OAuth login |
| Provider | State management |
| flutter_localizations | Internationalisation |
| Google Fonts | Typography |

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/fatmanagaa/evently_project.git
   cd evently_project
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

> Make sure you have Flutter installed and a device/emulator connected. Firebase is already configured in the project.
